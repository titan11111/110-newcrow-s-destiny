# CROW'S DESTINY — オープニング〜エンディング シーン一覧

オープニングからエンディングまで、ステージとボス戦を分けた一覧表です。

---

## メイン一覧表（面ごと・ステージ＋ボス戦）

各面は「ナレーション → ステージ（通常プレイ）→ ボス登場 → ボス戦 → ステージクリア」の順。7面クリアでエンディング。

| 面 | ステージ（PLAYING） | ボス戦（BOSS_FIGHT） | ステージの概要 |
|:--:|--------------------|----------------------|----------------|
| 1 | 灰燼の街道 | 穢れの先兵 — 彷徨う巨骸 | 異界の侵食が始まった地。灰に覆われた街道。 |
| 2 | 汚染された地下水路 | 粘体の母胎 — ハイヴコア | 毒に侵された水路。壁を這う穢者が闇から迫る。 |
| 3 | 封印研究所 | 擬態する知性 — ミミック | 穢れを解き明かそうとした場所。今は穢れに飲まれた廃墟。 |
| 4 | 崩落の高架 | 蒼穹の守護者 — 鉄翼 | 空は異界の門に覆われ、風が刃のように吹き荒ぶ。 |
| 5 | 墜ちた方舟の内部 | 門番 — 多脚のガーディアン | 異界から落ちた巨大な残骸。有機と無機が融合した迷宮。 |
| 6 | 培養層・蒼の深淵 | 蒼穢の女王 — ブルーコア | 無数の蒼光が脈打つ培養の間。穢れの源が近い。 |
| 7 | 次元の裂け目 | 裂け目そのもの — ヴォイド（ラスボス） | 現実が軋む。世界の境界が消える。全ての根源がここにある。 |

## 通しシーン一覧（オープニング→エンディング）

| # | シーン | 状態（state） | 内容 |
|:--:|--------|---------------|------|
| — | **取り扱い説明** | INSTRUCTIONS | アイテム（聖遺物）のアイコンと説明。下の「START」でオープニングへ。 |
| 0 | **オープニング** | TITLE | タイトル画面。オープニングBGM再生。「START」で儀式開始。 |
| 1 | ナレーション | NARRATION | ステージ名・説明表示。スキップ可。 |
| 2 | **ステージ** | PLAYING | 通常プレイ（スクロール・雑魚・蒼穢3体でボス出現） |
| 3 | ボス登場 | BOSS_INTRO | ボス登場演出（約120フレーム） |
| 4 | **ボス戦** | BOSS_FIGHT | ボス戦 |
| 4' | ラスボス 第二→第三形態移行演出 | LAST_BOSS_2TO3_CUTSCENE | 7面のみ。第二形態撃破後、lastbossadvance2to3.png を左→右ワイプ＋コメント表示。終了後第三形態（猫神）戦開始。 |
| 5 | ステージクリア | STAGE_CLEAR | 「STAGE CLEAR」表示。次面へ or エンディングへ。 |
| — | （2〜7面） | 上記 1〜5 を繰り返し | |
| 終 | **エンディング** | VICTORY | 全7面クリア。「浄化の儀式、完遂せり」。タイトルへ戻る。 |

※ プレイ中・ボス戦中にHP0で **GAME_OVER** → タイトルへ。

## 状態（state）対応表

| state | シーン |

| INSTRUCTIONS | 取り扱い説明（アイテム説明・STARTでオープニングへ） |

| GAME_OVER | ゲームオーバー |
| VICTORY | エンディング（全クリア） |
