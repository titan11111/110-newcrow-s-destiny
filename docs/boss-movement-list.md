# ボス動き一覧

全ボスは表示・当たり判定が **60%** に統一。その代わり画面内を動き回り、それぞれ個性のある軌道になる。

---

## 1. 穢れの先兵 — 彷徨う巨骸（Boss1）

| 項目 | 内容 |
|------|------|
| **通常** | 画面中央を軸に **左右スウィング**（幅 `W×0.38`）。端で反転。縦は `ty` を中心に二重 sin で漂う（振幅 H×0.22 / H×0.08）。 |
| **範囲** | 横: `centerX ± moveRange`（centerX = W×0.5, moveRange = W×0.38）／縦: `clamp(y, H×0.12, H×0.55)` |
| **個性** | 巨骸らしい「重厚な左右往復＋ゆっくり上下」の軌道。 |

---

## 2. 粘体の母胎 — ハイヴコア（Boss2）

| 項目 | 内容 |
|------|------|
| **通常** | `tx`/`ty` を中心に **横スライド**（sin×slideRange + cos×W×0.12）、縦は二重 sin（H×0.2 / H×0.08）。slideRange = **W×0.42**。 |
| **範囲** | 横: `clamp(x, W×0.12, W - W×0.12)`／縦: `clamp(y, H×0.1, H×0.5)` |
| **ピンチ** | **コア暴走**: ランダム目標点へ直線移動（12フレームごとに目標更新）。速度 6。 |
| **個性** | 母胎らしい「画面をゆっくりスライド＋上下に浮遊」する有機的な動き。 |

---

## 3. 擬態する知性 — ミミック（Boss3）

| 項目 | 内容 |
|------|------|
| **テレポート** | CD で **画面内ランダム** に瞬間移動。先に残像を追加。出現位置: `x ∈ [80, W-80]`, `y ∈ [70, H×0.6]`。ピンチで CD 短縮。 |
| **通常** | テレポート後は **浮遊**: sin/cos の二重で不規則な微変動（加算で x,y を少しずつずらす）。 |
| **範囲** | `clamp(x, 70, W-70)`／`clamp(y, 60, H×0.65)` |
| **個性** | 予測しづらい「テレポート＋不規則な漂い」。 |

---

## 4. 蒼穹の守護者 — 鉄翼（Boss4）

| 項目 | 内容 |
|------|------|
| **通常** | `tx`/`ty` を中心に **横 ampX = W×0.38**、**縦 ampY = H×0.22**。sin/cos で微変動（横に W×0.08 の cos 追加）。ピンチで横振幅 1.5 倍。 |
| **範囲** | 横: `clamp(x, 90, W-90)`／縦: `clamp(y, H×0.15, H×0.6)` |
| **特殊** | **ダイブボム**: 自機真上へ移動後、直線急降下。**スクラップストーム**: 急降下＋横に flyDir で移動。**デスロール**: sin/cos で軌道を描きつつ炎弾を残す。 |
| **個性** | 空域を大きく横断し、上下にバンクする「蒼穹の守護者」らしい広い飛行。 |

---

## 5. 門番 — 多脚のガーディアン（Boss5）

| 項目 | 内容 |
|------|------|
| **通常** | `tx`/`ty` を中心に **横 W×0.36×moveMul**、**縦 H×0.2 + H×0.06** の二重 sin。moveMul はピンチで拡大。 |
| **範囲** | 横: `clamp(x, 80, W-80)`／縦: `clamp(y, H×0.12, H×0.55)` |
| **個性** | 門番らしい「一定範囲内で重厚に巡回」する動き。 |

---

## 6. 蒼穢の女王 — ブルーコア（Boss6）

| 項目 | 内容 |
|------|------|
| **通常** | `tx`/`ty` を中心に **横**: sin×（W×0.36）×（ピンチ時 1.5 倍）＋ cos×（W×0.1）。**縦**: 二重 sin（H×0.22 / H×0.07）。 |
| **範囲** | 横: `clamp(x, 75, W-75)`／縦: `clamp(y, H×0.1, H×0.58)` |
| **個性** | 氷の結晶が漂うような「優雅に画面を舞う」軌道。 |

---

## 7. 裂け目そのもの — ヴォイド（ラスボス・Boss7）

### 形態0・形態1

| 項目 | 内容 |
|------|------|
| **通常** | `tx`/`ty` を中心に **横** sin×（60×formMul）、**縦** cos×（35×formMul）。formMul: 形態0=1, 形態1=1.3。 |
| **個性** | 画面上部で脈動・ゆっくり漂う。 |

### 形態2（第3形態）

| 項目 | 内容 |
|------|------|
| **瞬間移動** | **voidTeleportCD** が 0 になると、現在位置を残像に追加し、**画面内ランダム**（x: 90～W-90, y: 80～H×0.6）にテレポート。CD は通常 75 フレ、ピンチ 45 フレ。テレポート後に `tx`/`ty` を更新。 |
| **残像** | テレポート時に **voidAfterimages** に `{ x, y, t: 180, frame }` を追加。毎フレーム `t` を減らし、約 **3 秒（180 フレーム）** かけてフェードアウト。本体と同じスプライト（60% スケール）で描画。 |
| **通常** | テレポート以外のフレームは、`tx`/`ty` を中心に **横** sin×（W×0.32）＋ cos×（W×0.08）、**縦** cos×（H×0.2）＋ sin×（H×0.06）。 |
| **範囲** | `clamp(x, 90, W-90)`／`clamp(y, 80, H×0.6)` |
| **個性** | 瞬間移動＋数秒残る残像で「次元を切り裂く」ような動き。 |

---

## 共通

- **登場時**: 画面右外（x = W+80）から `tx`, `ty` へゆっくり移動。到着で `arrived`。
- **表示スケール**: 全ボス `BOSS_SIZE_SCALE = 0.6`。当たり判定もこれに連動。
