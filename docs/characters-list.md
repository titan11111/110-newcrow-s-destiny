**共通仕様（雑魚）**

- **移動**: 一定速度で左方向（vx 約 -1.5〜-2.5）、縦は `baseY + sin(timer)*25` でゆらぎ。
- **攻撃**: 自機狙いの単発弾（青敵は5面以降でたまに2発目を拡散で追加）。
- **体力**: 通常 16×敵HP倍率、青 45×倍率。倍率は面が進むごとに上昇（1.0 → 2.6）。
- **スプライト**: ステージIDに応じて enemy2, enemy3, steam_wolf, mechanical_bat, enemy6, enemy7 を切り替え。4面＝スチームウルフ、5面＝メカニカルバット（いずれも3×3）。レイアウトは画像に依存。

## ゲーム進行（ボス戦への遷移）

- **ステージプレイ中（PLAYING）** に **蒼穢（青い敵）を 3 体撃破** すると **ボス戦（BOSS_INTRO → BOSS_FIGHT）** へ進む。画面上部の「蒼穢: n / 3」が 3 になると発動。
- 青い敵は一定間隔でスポーン（1体目は約 3〜5 秒後、2体目以降は約 5〜8 秒間隔）。倒すと「蒼穢 浄化 (n/3)」と表示され、n が 3 でボス登場。
