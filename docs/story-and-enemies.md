# CROW'S DESTINY — ストーリーと敵一覧

## ゲームのストーリー概要

異界の侵食が広がる世界で、**カラス**が穢れを浄化し、根源へと進む物語。  
各ステージは「侵食された場所」であり、ボスを倒すとその面のスキルを解放。  
**蒼穢（あおけがれ）**を3体倒すとボス戦へ。最終的に**次元の裂け目**の根源と対峙する。

## ステージ別ストーリー・敵・ボス

### 第1章：灰燼の街道（かいじんのかいどう）

| 項目 | 内容 |
|------|------|
| **舞台** | 異界の侵食が始まった地。灰に覆われた街道に穢れし者が蠢く。 |
| **通常敵（穢れし者）** | 紫系（#8888cc）。2種類の振る舞いからランダム。 |
| **蒼穢** | 青（#44aaff）。プレイヤー周囲を旋回し、距離を保って弾を撃つ。 |

#### 通常敵の振る舞い（1面）

| タイプ | 個性・動き | 攻撃パターン |
|--------|------------|--------------|
| **CHARGE** | 速い左移動、上下のゆらぎは小さい。突進気味。 | 自機狙い弾1発。 |
| **ZIGZAG** | 遅めの左移動、上下に大きいジグザグ。 | 自機狙い弾1発。 |

#### ボス1：穢れの先兵 — 彷徨う巨骸

| 項目 | 内容 |
|------|------|
| **キャラ** | 巨躯の骸。重量感のある左右往復＋縦の漂い。端で折り返すと衝撃・シェイク。 |
| **攻撃** | **骨弾バースト**（扇状に骨弾）／**骨の指弾**（追尾弾を3〜4発連続）／**テイルスウィング**（尾の薙ぎ払い＋骨破片9発）／**紫炎噴射**（前兆約2秒→自機方向にゆっくり追従する複数ビーム）／**突進**（自機に向かって突っ込み、軌跡に弾＋シェイク）。ピンチ時は**骨格崩壊散弾**。 |

| タイプ | 個性・動き | 攻撃パターン |
|--------|------------|--------------|
| **TELEPORT** | やや速い左移動。一定間隔で baseY を現在位置に更新（ポジションの“ジャンプ”）。 | 自機狙い弾1発。 |

#### ボス3：擬態する知性 — ミミック

| 項目 | 内容 |
|------|------|
| **キャラ** | 擬態する知性。画面内を不規則に浮遊し、**テレポート**で位置を変える。HP60%以下でテレポートCD短縮、30%以下で一定確率フェイクテレポート。 |
| **攻撃** | **テレポート**（画面内ランダムに移動）／**ノイズバレット**（不規則軌道の弾8発、テレポート直後に連動することあり）／**サンダーボルト**（前兆約1.5秒→直下に雷撃3本）／**データ侵食波**（横一列の波動を上下2段）／**ミラーダブル**（コピー1体、ピンチで2体。体力1、同じく弾を撃つ）。ピンチ時は**パラノイアフィールド**（画面全体に微細グリッチ弾）。 |

|--------|------------|--------------|

#### ボス4：蒼穹の守護者 — 鉄翼（Iron Wing）

| 項目 | 内容 |
|------|------|
| **キャラ** | 鉄の翼。3×2スプライトで PATROL→SPREAD→DIVE→SPIRAL→RECOVER をループ。HP50%以下で**RAGE**（移動・弾速・発射間隔が上昇）。 |
| **攻撃** | **PATROL**（波状に飛行しながら**3way弾**を定期的に発射）／**SPREAD**（扇状に9〜11発を3回）／**DIVE**（目標地点へ急降下→着地時に**8方向弾**）／**SPIRAL**（旋回しながら**2方向のスパイラル弾**）。RECOVER で上空に戻り次の PATROL へ。 |

## 蒼穢（青敵）まとめ

| 項目 | 内容 |
|------|------|
| **役割** | ボス戦の前哨戦。「格上」としてプレイヤー周囲を旋回し、**距離を保って**（最小半径150px）攻撃する。 |
| **移動** | プレイヤーを中心にした円軌道（半径150〜200px）。周期で少しだけ近づくが、近づきすぎない。 |
| **攻撃** | 自機狙い弾（青）。5面・6面では追加で1発（角度ブレ）を撃つ。 |
| **体力** | 通常敵の約2.8倍（45×ステージ倍率）。 |

## 予測射撃について

- **対象**：通常敵のうち、**4面・5面・6面・7面でスポーン時の50%**が `usePrediction = true`。
- **仕様**：直近30フレーム（約0.5秒）の自機位置を記録し、**最も古い3点の平均座標**を狙って弾を発射する。
- **効果**：同じ動きを続けると当たりやすくなり、慣れたプレイヤーにも回避の変化を促す。
