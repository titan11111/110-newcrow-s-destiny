# ボス動き＆演出強化 — 実装メモ

## 横断的共通

| 項目 | 実装内容 |
|------|----------|
| **断末魔フェーズ** | HP10%を切った瞬間に一度だけ **90フレーム（約1.5秒）完全停止**。`lastStandTriggered` で二重発火防止。 |
| **ダメージフィードバック** | 被弾で `hitFlash`＋HITモーション。Boss2は外殻デブリ、Boss5は `armorPeelLevel` を加算（描画で未使用のため今後の見た目変更に利用可能）。 |
| **FX拡張** | `floorCracks`（床ひび）、`arenaDarkCorners`（四隅暗転）、`arenaDebris`（デブリ）、`arenaFreeze`（画面端凍結）。`drawArenaEffects(c)` で一括描画。 |

---

## ボス別

### 1. 彷徨う巨骸
- HP50%以下で振り幅を **W×0.52** に拡大。
- **反転時**: シェイク35、衝撃波バースト、床ひび（45フレ）、画面上方から骨デブリ8個、重いSE（`_playBossSE big`）。

### 2. ハイヴコア
- HP30%以下かつコア暴走中、**15フレごとに軌跡を最大3本**（`rampTrail`）追加。描画で半透明の楕円として表示。
- **被弾時**: 外殻デブリを `fx.addArenaDebris` で3個追加（横方向に飛ぶ破片）。

### 3. ミミック
- HP60%以下でテレポートCD半減（通常75/ピンチ38）。
- HP30%以下で **20%フェイクテレポート**（残像のみ追加、位置はそのまま）。
- 実テレポート時: **ポータル残滓**を出現先に30フレ表示（破線円）、**四隅暗転**（`arenaDarkCorners` 45フレ）。

### 4. 鉄翼
- HP50%以下でダイブボムが **曲線追尾**（降下中に `px` へ向けて x を補間）。突入直前にシェイク22。
- デスロール中は **当たり判定1.3倍**（`hitRadius`）。
- HP15%以下で **撃墜寸前演出**（`outOfControlT`）: 火花のような短い線＋位置ジッター。

### 5. ガーディアン
- HP40%以下で **巡回速度1.8倍**＋高周波の小さな跳ね（sin/cos の追加項）。
- **エネルギーの残光**: 5フレごとに軌道を `energyTrail` に追加、最大24、120フレでフェード。描画で半透明の円弧。
- 被弾で `armorPeelLevel` 加算（0.25ずつ、最大3）。※描画での装甲剥離は未実装。

### 6. ブルーコア
- HP50%以下で **鋭角な折れ線混じり**の軌道（`angularPhase` と zig 項）。
- **氷の軌跡**: 4フレごとに `iceTrail` に追加、90フレでフェード。描画で青白い円。
- ピンチ時 **画面端凍結**（`fx.arenaFreeze = 60`、青白い縁で描画）。

### 7. ヴォイド（形態2）
- 瞬間移動の **残像は最大5本**（`voidAfterimages.length > 5` で shift）。
- **カオス軌道**: 複数の sin/cos を非整数倍で組み合わせた `chaos(a,b,c)` で、周期の揃わない予測困難な漂い。

## 未実装（要BGM・SE・アセット）

- **BGMのリアクティブ設計**（HP帯でレイヤー増減）: サウンドマネージャー側の対応が必要。
- **重低音「ドン」SE**（巨骸反転）: 現在は既存の `playBossBig`。専用SEを登録すれば差し替え可能。
- **コアの叫びSE**（ハイヴコア）: 専用SE未登録。
- **テレポート時のひっくり返ったSE**（ミミック）: 専用SE未登録。
- **鉄翼の風切り音・機体振動の常時SE**: ループSEの追加が必要。
- **ガーディアンの機械的警告音**（HP段階）: 専用SE未登録。
- **ブルーコアの半音下がり・不協和音**: BGM側の変更が必要。
- **形態2移行の無音の恐怖・星が吸い込まれる背景**: 背景／シーン演出の拡張が必要。
- **コントローラー振動**: ゲームpad API 対応が必要。

以上は必要に応じてサウンド／背景レイヤーを追加することで拡張可能。
