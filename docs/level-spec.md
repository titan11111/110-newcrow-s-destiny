# CROW'S DESTINY — レベル仕様の説明

ゲーム内で「レベル」が何に連動しているかを整理したドキュメントです。

---

## 1. ステージ番号（stageIdx）

- **範囲**: 0 〜 6（全7面）
- **連動するもの**:
  - **ステージデータ**（`js/data/stages.js` の `STAGES[stageIdx]`）
    - ステージ名・説明文・空/地面の色
    - 敵の体力倍率（enemyHpMul）、弾速（enemyBulletSpd）、出現間隔（spawnMin/Max）
    - ボス名・ボス色・ボス基本体力（bossHpBase）・攻撃速度（bossAtkSpd）
  - **背景タイプ**（bgType）: ROAD / SEWER / LAB / BRIDGE / ARK / HIVE / VOID
  - **障害物の色**（obsColor, obsGlow）
  - **BGM**: 通常プレイ時は `stage1` 〜 `stage7`、ボス戦は `boss` またはラスボス用
- **変化タイミング**: ステージクリア後、次面へ進むとき `stageIdx++`

---

## 2. 武器レベル（weaponLevel / カラスの攻撃力）

- **範囲**: 1 〜 6
- **レベル閾値（スコア）**:
  - Lv.1: 0 〜 9999
  - Lv.2: 10000 〜 24999
  - Lv.3: 25000 〜 54999
  - Lv.4: 55000 〜 79999
  - Lv.5: 80000 〜 99999
  - Lv.6: 100000 〜
- **連動するもの**:
  - **弾の発射間隔**: `intv = max(4, 14 - weaponLevel * 2)`（レベルが高いほど連射が速い）
  - **通常弾**: レベル1＝1本、レベル2〜5＝2〜5本（扇状に広がり）
  - **Lv.6 追加**: **ギャラクシー砲** — 手前に光る球が出て一直線にレーザー。青白い光（浄化の青いほむら）。当たり判定・ビーム扱い（高ダメージ・判定幅やや大）
  - **ボスへのダメージ**: 通常弾 `6 + weaponLevel`、ビーム/ギャラクシー `10 + weaponLevel`
  - **雑魚へのダメージ**: 通常弾 `8 + (weaponLevel - 1) * 2`、ビーム/ギャラクシー `14 + (weaponLevel - 1) * 2`
- **変化タイミング**: スコアが上記閾値を超えたフレームでレベルアップ（「LEVEL UP!」表示＋SE）

---

## 3. ボス側の「フェーズ」と「idx」

- **idx**: 実質ステージ番号（0〜6）。ボスの攻撃パターン・振幅・弾数・フェーズ数が `idx` に比例して強化される。
- **フェーズ（phase）**: ボス戦中の攻撃パターン（全方位弾 / 突進バースト / 狙い弾＋召喚 / スパイラル / レーザーなど）。`phaseDur` 経過で次のフェーズに切り替わる。
- **maxPhases**: `3 + min(idx, 2)` で、ステージ1は3フェーズ、ステージ2〜は5フェーズ。

---

## まとめ

| 用語 | 何を指すか | 主な影響 |
|------|------------|----------|
| **ステージ / 面** | stageIdx（0〜6） | 難易度・見た目・BGM・ボス |
| **レベル**（HUD等で言う場合） | weaponLevel（1〜6） | カラスの攻撃力・連射数・Lv.6でギャラクシー砲 |
| **ボスの難易度** | idx + phase | 弾の数・速度・動きの振幅 |

※ HUD に表示している「レベル」が weaponLevel を指すか、ステージ番号を指すかは、表示ロジック（`js/render/hud.js`）を確認してください。
